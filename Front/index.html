Informações Técnicas

Object Reader

Versão 1.0.0

Sistema de gestão de patrimônio escolar com leitura de código de barras via câmera, desenvolvido para uso interno e integração com o núcleo de educação.

Stack Tecnológico
  • HTML5 — Estrutura semântica
  • CSS3 — Grid, Flexbox, Custom Properties
  • JavaScript — Vanilla JS, sem frameworks
  • BarcodeDetector API — Leitura nativa
  • Lucide Icons — Ícones SVG
  • Google Fonts — Syne + DM Sans
Compatibilidade
  • Google Chrome (recomendado)
  • Microsoft Edge
  • Android — Chrome
  • Firefox — BarcodeDetector não suportado
  • Safari — BarcodeDetector não suportado
Formatos Suportados
Code 128
EAN-13
Code 39
QR Code
Data Matrix
PDF417
Arquitetura do Projeto
index.html
Estrutura unificada da aplicação. Páginas renderizadas via JS com show/hide por data-page. Layout responsivo via CSS.
style.css
Estilos organizados por seções com CSS Custom Properties para theming e media queries para responsividade mobile/desktop.
script.js
Lógica de navegação SPA-like, BarcodeDetector API, gerenciamento de câmera e renderização dinâmica de listas.
